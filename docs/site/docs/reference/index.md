# Reference

This section is the alphabetical / structured reference material. It is meant to be skimmed and grepped, not read top to bottom. Each page in this section is a flat list of fields, flags, or terms with their meaning and default.

Topics covered here include the full `rastreo` and `rastreo-server` CLI flag reference, the scenario YAML field schema (every key under `defaults` and `scenarios`), an error reference mapping each `RastreoError` variant to its meaning and likely cause, and a glossary of network-discovery terminology (OUI, CIDR, identity key, fuser confidence, sysObjectID).
